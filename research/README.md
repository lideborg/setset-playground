## Research Architecture

### 3-Layer System

**Layer 1: Aesthetic Foundation** (Aesthetics Research)
- Define visual language for each aesthetic
- Provide AI image generation keywords
- Create reusable aesthetic profiles

**Layer 2: Prompt Engineering** (System Guides)
- 9-component structured prompt template
- 6-dimensional variation system
- Systematic variation methodology

**Layer 3: Talent Management** (Talent & Models + Location Planning)
- 25 diverse talent models
- 20-prompt per model system
- Similar-but-different location mapping

### Critical Integration Points

**BRAND_RENAMING_STRATEGY.md** → Bridges aesthetic research and prompt building

**PROMPT_BUILDING_GUIDE.md** + **VARIATION_STRATEGY_GUIDE.md** → Translate visual analysis into structured prompts

**LOCATION_MAPPING_SYSTEM.md** + **EDITORIAL_MINIMAL_SYSTEM.md** → Operational constraints unifying all models

## Usage Workflows

### For AI Prompt Generation
1. **Start**: PROMPT_BUILDING_GUIDE.md + VARIATION_STRATEGY_GUIDE.md
2. **Reference aesthetics**: Pull from 01_AESTHETICS_RESEARCH based on desired style
3. **Ensure consistency**: Use TALENT_ROSTER.md for model characteristics
4. **Plan locations**: Consult LOCATION_MAPPING_SYSTEM.md for assignments

### For New Brand Analysis
1. **Use frameworks** from French/Italian/Minimalist files as templates
2. **Document**: Photography style, lighting, color palette, mood, styling
3. **Create AI keywords** for prompt generation
4. **Apply naming strategy** from BRAND_RENAMING_STRATEGY.md

### For System Maintenance
1. **Update**: FINAL_UPDATES_SUMMARY.md as changes are made
2. **Track**: Variation matrices in VARIABLE_PROMPT_SYSTEM.md
3. **Verify**: Location diversity against LOCATION_MAPPING_SYSTEM.md
